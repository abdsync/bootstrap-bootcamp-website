# Frontend Bootcamp Website

## Overview
This repository contains the source code for a responsive "Frontend Bootcamp" website. The project serves as a practical implementation of fundamental frontend web development principles, primarily utilizing the Bootstrap 5 framework to achieve a mobile-first, responsive design. 

**View Online:** https://bootcamp-vert.vercel.app/

## Technical Features
The application integrates several core Bootstrap 5 building blocks to manage content and user interaction:
* **Grid System**
* **Navbar**
* **Cards**
* **Accordion**
* **Modal**
* **Forms & List Groups**
